各授業回の学習事項

第1回 WWW と HTML

HTMLは数々の「要素」で作られているということを学んだ。その要素にはブロック要素とインライン要素があり、インライン要素はブロック要素の内側でなければならないなどの決まりがあることを知った。また、要素の種類によってできることが異なる。例えば、リスト要素で箇条書きのようなレイアウトにすることができる。このような要素は数え切れないほどの種類を調べるにあたってMDN Web Docs上で調べることが大切だということを学んだ。

演習の成果:演習1-3:HTMLの文書の作成

演習の成果:演習1-4:HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

CSSでページレイアウトを考え装飾した。テキストサイズの変更やテキストボックスを作り、そのボックスの中に画像を入れたり枠を作った。注目されるべきである一番大きな文字に対して影をつけるなどの工夫を入れることで強調した。

演習の成果browser-intro.html

第3回 JavaScript の基本

JavaScripsはインタプリタ言語でコンパイル言語のJavaとは名前が似ているが別物になっている。Javaにはコンパイルという一手間が必要だが、JavaScriptにはそれが必要ないという利点がある一方で、エラーになりづらかったりエラーに気づきづらいというデメリットもある。授業ではコンソール画面でJavaScriptで書いたコードを表示させることをした。その際のJavaとの違いは、複雑な型の指定をせずに全てletにしたり、関数の書き方がスッキリしていて見やすかったりしていて人間に見やすい言語だと思った。エラーについてはエラーにならないプログラムが存在していて、間違った式の計算結果は全てNaN(Not a Number)と表示される。これはエラーではないのでプログラムは停止することなく実行され続けてしまう。他にundefined(未定義)やnull(空っぽ)があることを学んだ。

演習の成果:obj-practice.html
